personal / select works

here is a collection of work i've created over the past decade. i have never tried to contain, bind, corner, or limit myself to a singular way of creating work. i've pulled together this collection because i like to think it represents that lack of constraint, and i hope it shows a few different parts of my being. i would like to add more, but i made this website for a web development class and could not commit more time to do so. i should note that i include any form of time limitation in my definition of "a singular way of creating work."

thanks for visiting.
